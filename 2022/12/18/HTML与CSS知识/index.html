CSS选择器与常用属性

CSS选择器

CSS语法：

​ 选择器{

​ 属性名：属性值

​ }

style标签：

​

常用的css选择器：
元素选择器：直接通过标签名来寻找

​ h1{xxx}

类选择器：加入class属性（指定多个同class名的元素）

​ .class名{xxx}

id选择器：加入id属性（指定单个元素）

​ #id名{xxx}

通配符选择器：

​ *{xxx}

层级选择器：选择某一class属性下的某一标签

​ .class属性名 h1{xxx}

组合选择器：用逗号分隔

​ 选择器1,选择器2{xxx}

伪类选择器：增加行为

​ 选择器：hover{xxx}

​ hover：鼠标指上的时候的选择

伪元素选择器：增加元素

​ 选择器::befor，选择器::after

​ 在元素之前加内容/在元素之后加内容

选择器的权重：

​ 相同选择器:后面的选择器会覆盖前面的选择器

​ 不同选择器：id选择器的权重 > class选择器的权重 > 元素选择器的权重

​ 层级选择器：计算多个选择器权重

​ ！important属性 让权重设为最高

常用的css属性：

​ font-size:字体大小

​ （Chrome浏览器中font-size最小值是12px）

​ color：字体颜色

​ width：宽度

​ height：高度

​ 高度宽度可能强行拉伸图片

​ background：背景

​ text-align：文本水平居中

​ line-height：文本行高（垂直居中）

引入css的方法：

嵌入式：

​ （少量的样式）

内联式：

​ 写在属性里面style（不常用）

​ 内联样式行内样式权重高于嵌入样式

外部式：

​ 引入外部css文件（常用）

​ 方便拆分为两个窗口

​ link标签引入外部css文件

盒子模型：

将HTML元素看作盒子实现的网页布局

有关盒子模型的css属性：

​ 边框：border

​ border-width：边框宽度

​ border-style：边框形式

​ none无边框 / dashed虚线边框 / solid实线边框 / double双线边框 / dotted点状边框

​ border-color：边框颜色

​ 外边距：margin

​ top / right / bottom / left

​ 4个值的时候上右下左、2个值的时候上下、左右

​ 内边距：padding

​ 设置距离会改变盒子大小

认情况下元素宽度和高度的计算

​ 元素实际宽度 = border-left + border-right + width + padding-left + padding-right

​ 元素实际高度 = border-top + border-bottom + height + padding-top + padding-bottom

​ 设置属性：box-size:border-box

​ 元素实际宽度 = width

​ 元素实际高度 = height

​ 元素边距初始化（清除系统默认边距）

​ *{margin:0px;padding:0px}

​ 元素布局：margin：auto元素自动居中

列表的样式：

​ 取消列表样式：list-style：none

​ 列表样式在边距之内：list-style:inside

HTML元素的分类：

​ 块元素：可以设置高度宽度，独立成行。

​ a、span

​ img、input、button

​ inline-block：转换为行内块元素

CSS定位

css定位（position）

绝对定位：absolute

​ 脱离文档流（不占位）

​ 默认参照为浏览器视窗的左上角

相对定位：relative

​ 不脱离文档流（占位）

​ 默认参照为原本所在的位置

固定定位：fixed

​ 脱离文档流（不占位）

​ 默认参照为浏览器视窗的位置

坐标属性：非定位元素不起作用

媒体查询

flex布局

​ 让网页布局更简洁，更易于维护

基本概念

​ 让元素设为display:flex;元素变为一个flex容器

​ 容器内部的元素为flex元素或者flex项目

​ （display：none隐藏)（display：block块元素）（display：inline行元素）

flex容器中的默认效果

​ flex项目在flex容器中沿主轴排列

​ flex项目高度适应容器高度（同行内元素）

设置flex容器
flex-direction:设置flex项目排列方向

​ row:默认值，主轴为水平，起点在左端

​ row-reverse:主轴为水平，起点在右端

​ column:主轴为垂直，起点在上端

​ column:-reverse:主轴为垂直，起点在下端

justify-content:flex项目主轴排列方式

​ flex-start(默认值)：左对齐

​ flex-end:右对齐

​ center:居中

​ space-between:两端对齐，项目之间的间隔都相等

​ space-around:每个项目两端间距相等。项目间隔比项目与边框之间间隔大一点

align-items:flex项目在交叉轴的排列方式

​ flex-start：交叉轴的起点对齐

​ flex-end:交叉轴的终点对齐

​ center:交叉轴的中点对齐

​ strech:(默认值)：如果项目未设置高度或设为auto，将占满整个容器高度

flex项目

​ flex-grow:属性定义项目的放大比例,默认为0 ,空间充足,等比例补全。+

​ flex-shrink:定义了项目的缩小比例,默认为1 ,即如果空间不足,该项目将缩小

​ flex-basis：主轴排列为宽度,交叉轴排列为高度,设置px ,默认值是auto。

​ flex：综合上面三个样式

​ align-self:flex项目的对其方式( auto | flex-start | flex-end | center | baseline | stretch )

grid布局

flex布局是一维布局，gird是二维布局。

flex布局考虑的是项目按照行或者列布局，grid布局需要同时考虑行和列。

属性：

  • grid-tempalte-columnss：设置行宽
  • grid-tempalte-rows：设置行高
    • px具体的像素
    • fr按照比例占满整个容器
  • grid-auto-flow:column 元素纵向排列

容器属性：

​ 在单元格中的对齐方式

  • justify-items 属性

  • align-items 属性

    • 默认stretch（延伸）
    • start开始
    • center居中显示
    • end末尾

    单元格在容器中的对齐方式

  • justify-content属性

  • align-content属性

    • center单元格位置在容器中居中

    设置溢出的尺寸

  • grid-auto-columns属性

  • grid-auto-rows属性

项目属性：

​ 合并单元格横向

  • grid-column-start属性

  • grid-column-end属性

  • grid-column属性

    合并单元格纵向

  • grid-row-start属性

  • grid-row-end属性

  • grid-row属性

    设置字体位置

  • justify-self属性

  • align-self属性

同一套静态页面代码在不同设备中展现出不同的效果，适应多个设备
